# My Blog

This project involves a blog where a user can login, add post, comment and like other user's post

## Built With

- Ruby
- SQL

## Getting Started

- Clone repository [link](https://github.com/Sevenpros/My-blog).
- Change directory into project
- Run `ruby main.rb`
